# Crash Course: Raspberry Pi Pico / Embedded C

Ein praxisorientierter Kurs für Einsteiger in die Embedded-Entwicklung mit dem Raspberry Pi Pico.
Zielgruppe sind Entwickler mit Java-Kenntnissen, die in die Welt der Mikrocontroller und C-Programmierung einsteigen möchten.

## Inhalte

- WSL2 & Linux-Grundlagen
- Git-Grundlagen
- C für Java-Umsteiger
- *(weitere Kapitel in Arbeit)*
